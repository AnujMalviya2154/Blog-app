# Blog Web Application

A full-stack blog application built using Node.js, Express, and EJS that allows users to create, edit, and delete blog posts dynamically.

## Features
- Create, edit, and delete blog posts
- Dynamic content rendering using EJS
- RESTful routing
- Clean UI and structured backend

## Tech Stack
- Node.js
- Express.js
- EJS
- HTML, CSS

## Project Structure
- routes/ → Handles routing logic
- views/ → EJS templates
- public/ → Static files (CSS, images)

## How to Run Locally
1. Clone the repository:
   git clone https://github.com/AnujMalviya2154/Blog-app.git

2. Install dependencies:
   npm install

3. Run the server:
   node app.js

4. Open in browser:
   http://localhost:3000
